## Project Overview  

**QuickQuiz** is a full-stack quiz application designed to make learning interactive and engaging.  
It allows users to attempt quizzes, view instant results, and test their knowledge through a modern, responsive web interface.  

The project features a **Spring Boot backend** that manages quiz data and REST APIs, and a **JavaScript-based frontend** for seamless interactivity.  

---

## Key Features  

✅ Interactive multiple-choice quizzes  
✅ Real-time score calculation and feedback  
✅ RESTful APIs for data communication  
✅ MySQL database with Spring Data JPA  
✅ Responsive UI using HTML, CSS & JavaScript  
✅ Easy to extend — add new subjects or questions easily  
✅ Simple integration for deployment and scaling

## Architecture  

Frontend (HTML, CSS, JavaScript)
↓ REST API
Backend (Spring Boot)
↓
Database (MySQL)

This modular structure ensures scalability, maintainability, and a clean separation of concerns.

## Getting Started  

### 1.Clone the Repository  
```bash
git clone https://github.com/Uzmaalia/QuickQuiz.git
cd QuickQuiz
```
### 2.Setup Backend
mvn clean install
mvn spring-boot:run

### 3.Run Frontend
Open the index.html file in your browser, or serve it locally using:
npx live-server
